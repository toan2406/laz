Laz is an attempt to bring lazy evaluation into Javascript using macros.

## Setup

- Install dependencies

```shell
yarn add babel-plugin-macros laz -D
```

- In your Babel config

```json
{
  "plugins": ["babel-plugin-macros"]
}
```
